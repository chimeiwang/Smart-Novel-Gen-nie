# 历史归档

本目录中的文档只保留历史排查、旧方案、旧计划和临时工作记录。

规则：

- 归档文档不作为当前实现依据。
- 与当前代码、schema、测试、`DOCS.md`、`AGENTS.md` 或 `src/agents/AGENTS.md` 冲突时，以当前事实为准。
- 如果归档文档中仍有有效规则，必须迁入当前权威文档；不要让开发者回归档里找规则。

目录：

| 目录 | 内容 |
| --- | --- |
| `agent-history/` | Agent 历史长 changelog 和旧说明 |
| `legacy-plans/` | 旧蓝图、旧方案、旧审计、历史问题记录 |
| `legacy-requirements/` | 旧 `X.X` 需求追溯碎片 |
| `superpowers/` | 旧 superpowers specs / plans |
| `working-notes/` | 根目录迁出的临时计划、发现和进度记录 |
